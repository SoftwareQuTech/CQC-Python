Examples using the python library
=================================

The examples below assume you have a background running which can read the CQC messages being sent.
To get started it is recommend to use `SimulaQron <http://www.simulaqron.org/>`_ and to first go through https://softwarequtech.github.io/SimulaQron/html/GettingStarted.html.
Further examples can also be found at https://github.com/SoftwareQuTech/CQC-Python/tree/master/examples.
